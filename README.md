# dxf-analyzer1
Extrator de dados de desenhos técnicos DXF para automação industrial.

## Funcionalidades
- Contagem de entidades (linhas, círculos, polilinhas)
- Cálculo do comprimento total de linhas/polilinhas

## Tecnologias
- Python 3
- ezdxf
